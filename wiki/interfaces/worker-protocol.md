# Pyodide worker protocol

M4 introduces a dedicated module Web Worker for all Python execution. The
worker is the only browser component allowed to import Pyodide or execute
`microtubes_core`.

## Asset loading

Runtime assets are same-origin static files generated by
`scripts/prepare_pyodide_assets.mjs`:

- `public/vendor/pyodide/` from the pinned NPM `pyodide` package;
- `public/wheels/microtubes_core-0.1.0-py3-none-any.whl` from `uv build`.

ADR-0006 records the hosting decision.

## Messages

All envelopes include `protocol_version: "1.0.0"` and `request_id`.

Main thread to worker:

- `init`: load Pyodide, NumPy, Pydantic, and the local wheel.
- `compute`: validate and evaluate one `SimulationRequest`.
- `cancel`: mark a request as cancelled; the worker suppresses delivery if the
  cancellation is observed before result posting.

Worker to main thread:

- `progress`: stage and short status text.
- `ready`: initialization completed.
- `result`: `SimulationResultPayload` plus transferable `ArrayBuffer`s.
- `cancelled`: request result will not be delivered.
- `error`: worker-level error code and detail.

## Array transport

The Python API returns C-contiguous `float64` NumPy arrays. The worker obtains
Pyodide buffer views, copies each array into a dedicated `Float64Array`, and
transfers the backing `ArrayBuffer` to the main thread. The JSON payload keeps
`GridFieldRef.buffer_index` as the link from metadata to the transferred array.

## Caching and supersession

The frontend client computes a local SHA-256 cache key over a stable JSON
serialization of `SimulationRequest`. This cache key is for client reuse only;
the authoritative scientific `request_hash` remains the Python core hash in
`SimulationResultPayload`.

When a new compute request starts, the client sends a cancellation message for
the previous active request and rejects stale cancelled requests. Pyodide
execution is single-threaded inside the worker, so cancellation is deterministic
at delivery boundaries rather than an interrupt of already-running Python code.
