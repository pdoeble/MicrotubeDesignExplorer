## [0.1.0] - 2026-07-11

### Added

- M0 repository and governance baseline: fixed layout, toolchain
  (pnpm + uv with lockfiles), wiki structure, ADR and plan templates,
  CI with prohibited-file check.
- M1-M3 scientific baseline: MATLAB inventory, immutable golden references,
  versioned contracts/defaults, and Python scientific-core parity.
- M4-M7 browser application: Pyodide worker execution, two-cooler input
  workflows, Plotly result maps, figure export, standalone HTML/PDF reports,
  and canonical JSON sidecars.
- M8/M9 release preparation: acceptance tests, accessibility/performance
  evidence, Pages workflow, deployed smoke configuration, and strict release
  gate checks.
